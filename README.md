### Type
1. integer is default int, and float is default double
1. L should be added after numbers when expressing long type; f should be added expressing float
1. int, double, long, byte, char are mainly used types (char occupies two bytes in java)
1. all types are reference type except for basic types.

### 命名规范
### naming style
1. 变量小驼峰，名词
1. variable small-camel, noun
1. 函数小驼峰，动词
1. function small-camel, verb
1. 类名大驼峰，名词
1. class big-camel, noun
1. 常量全大写，下划线分隔
1. constant all-uppercase, underscore split
1. 包名全小写
1. package all-lowercase
- 变量名有意义
- naming meaningfully

### 字符编码
### symbol character
- 一个字节表示ASCII，两个字节表示Unicode
- 1 byte represents ASCII, 2 bytes represent Unicode
- 8位二进制能表示ASCII，16位二进制能表示Unicode
- 8 bits represent ASCII, 16 bits represent Unicode
